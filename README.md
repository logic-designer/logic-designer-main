# Logic Designer

Logic Designer is a browser-based digital logic design and simulation tool.

## Development Preview

This repository currently contains **development preview builds** of the application.

The file provided here is a **compiled standalone HTML build** intended for testing and evaluation.

The full source code and modular development version are still under active development and may be released later.
